# Использованный стек
- Архитектура - MVVM (+ использование дополнительного слоя Router)
- UI - SwiftUI
- Работа с сетью - URLSession
- Асинхронность - async / await
- Безопасное хранение данных - Keychain
- Загрузка картинок - SDWebImage
- Менеджер зависимостей: SPM
